# VisualPlayer

<img src="resources/icon.svg" alt="VisualPlayer" width="112" />

**ほぼすべての形式を再生する、軽量で非営利のクロスプラットフォーム・メディアプレーヤー。**

`Rust · Tauri 2 · Swift` ｜ `Windows · macOS · Linux · Android · iOS`

[English](./README.md) ｜ [한국어](./README.ko.md) ｜ 日本語

</div>

---

VisualPlayer は **mpv（libmpv・内部 FFmpeg）** によりほぼすべての動画・音声形式を再生します。
複数再生、同名字幕の自動読み込み、トレント／ウェブ／YouTube リンクファイル、キーボード
ショートカット、タッチ操作に対応した**無料のオープンソース**アプリです。

## インストール

[最新リリース](../../releases/latest)からお使いのプラットフォーム向けファイルを入手してください。

| プラットフォーム | ファイル |
| --- | --- |
| Windows 10/11 (x64, Arm64) | `VisualPlayer-Setup-*.exe`（カスタムインストーラー） |
| macOS (Apple Silicon / Intel) | `.dmg` |
| Linux | `.deb` / `.rpm` / `.AppImage` |
| Android | `.apk` |

- **Windows:** `VisualPlayer-Setup-*.exe` を実行し、フォルダー・ショートカット・
  ファイル関連付けを選んでインストール。NSIS 不要の単一インストーラーで、
  アンインストールはプログラムの追加と削除から。
- **macOS:** アプリケーションへドラッグ。初回は右クリック →**開く**。
- **Linux:** パッケージを導入、または `chmod +x *.AppImage && ./VisualPlayer.AppImage`。
  システムの `libmpv` と WebKitGTK が必要です。
- **Android:** 提供元を許可して `.apk` を開きます。

## 使い方

1. **メディアを開く** — ウィンドウにファイルをドラッグ＆ドロップ、または右上の
   **フォルダー**ボタン／開始画面の**ファイルを開く**。
2. **複数ファイル**をドロップするとキューに追加され、最初の項目が再生され残りは
   プレイリストに入ります（**リスト**ボタンまたは `Tab` で表示）。
3. 前回の位置から**自動的に再開**します（設定でオフ可能）。

## 再生コントロール

再生中に自動的に隠れるコントロールバー（マウスを動かすと表示）：

- 再生／一時停止、前へ／次へ、時間付きシークバー、音量・ミュート、再生速度。
- **字幕**ボタンで字幕トラックを切り替え。
- **調整**ボタンで映像・音声ツールパネルを表示。
- **スクリーンショット**、**プレイリスト**、**設定**、**全画面**ボタン。

## 字幕

- **自動** — `movie.mkv` を開くと、同じフォルダーの `movie.srt`、`movie.ass`、
  `movie.ja.srt` などを自動的に読み込みます。
- **手動／内蔵** — 字幕ボタンまたは `c` でトラックを切り替えます。
- **同期・スタイル** — 調整パネルで**字幕の遅延**（`g`/`h`）、設定で**サイズ**を調整します。

## 調整（VLC 風）

**調整**パネルの機能：

- **映像:** アスペクト比、回転、ズーム、インターレース解除。
- **カラーイコライザー:** 明るさ・コントラスト・ガンマ・彩度・色相（＋リセット）。
- **音声:** トラック切替、**音声の遅延**（`Shift+G`/`Shift+H`）。
- **字幕:** トラック切替、遅延。
- **A-B リピート:** A を指定し B を指定すると区間をループ（`r`）。
- **メディア情報:** 解像度、コーデック、フレームレート、トラック数。

## キーボードショートカット

| 操作 | キー |
| --- | --- |
| 再生 / 一時停止 | `Space`, `k` |
| シーク ±5秒 / ±60秒 | `←` `→` / `Shift+←` `Shift+→`（`j` `l` も可） |
| フレーム送り | `.` / `,` |
| 音量 + / − / ミュート | `↑` / `↓` / `m` |
| 全画面 / 終了 | `f` / `Esc` |
| 速度 + / − / リセット | `]` / `[` / `Backspace` |
| 前 / 次の項目 | `p` / `n` |
| スクリーンショット | `s` |
| 字幕 / 音声切替 | `c` / `b` |
| 字幕遅延 − / + | `g` / `h` |
| 音声遅延 − / + | `Shift+G` / `Shift+H` |
| プレイリスト / 設定 | `Tab` / `Ctrl+,` |
| A-B リピート | `r` |
| アスペクト / 回転 | `a` / `Ctrl+R` |
| ズーム + / − / リセット | `=` / `-` / `0` |
| 先頭 / 末尾へ | `Home` / `End` |

完全な一覧は **設定 → キーボードショートカット**に表示されます。

## タッチ操作（モバイル）

Android（および iOS）で映像の上：

- **横スワイプ** — シーク（目標時間を表示）。
- **縦スワイプ・右半分** — 音量。
- **縦スワイプ・左半分** — 画面の明るさ。
- **左右ダブルタップ** — −10秒 / +10秒。

**設定 → タッチ操作**で有効・無効を切り替えられます。

## カスタムファイル形式

- **`.{動画}.torrent` + `.{字幕}.torrent`** — 動画トレントを開くと、同名の字幕トレント
  （`movie.mp4.torrent` ↔ `movie.srt.torrent`）があれば両方をダウンロードして字幕を
  動画に付加します。ダウンロード専用。
- **`.webvideo`** — 直接のメディア URL（1 行）、または `url`・`headers` を含む TOML。
- **`.ytvideo`** — YouTube リンク（1 行）、または `url`・`preferred_resolution`・`title` の TOML。

ダブルクリックで再生します。

## 既定のプレーヤーに設定

**設定 → 既定のプレーヤーに設定**で、対応するすべての動画・音声・字幕形式の既定アプリ
として VisualPlayer を登録します（OS の仕組みを使用。Windows では既定のアプリ設定を開きます）。

## 設定

言語（English / 한국어 / 日本語）、テーマ（システム/ライト/ダーク）、再開、ハードウェア
デコード、次を自動再生、字幕の自動読み込み・サイズ、ジェスチャー、既定アプリ登録、
ショートカット一覧、バージョン情報。

## ソースからビルド

前提：Rust(stable)、Node.js(LTS)、各プラットフォームの Tauri 前提条件。実際の再生には
**mpv/libmpv** が必要です。

```bash
# Linux ビルド依存 (Debian/Ubuntu)
sudo apt-get install -y libwebkit2gtk-4.1-dev libmpv-dev librsvg2-dev libgtk-3-dev

# macOS
brew install mpv

# 純粋コアのテスト（ネイティブ依存不要）
cargo test -p vp-core

# デスクトップ開発 / ビルド
cd app/main
npm install
npm run tauri dev
npm run tauri build               # トレント対応は `-- --features torrent`

# Android (Android SDK/NDK が必要)
npm run tauri android init
npm run tauri android build
```

バックエンドは Cargo **ワークスペース**です。`vp-core` は依存のない単体テスト済みロジック、
`app/main/src-tauri` は Tauri バイナリです。フロントエンドは **Vite + Vanilla TypeScript**
（フレームワークなし）。iOS は [`app/ios`](./app/ios) の独立した Swift アプリです。
