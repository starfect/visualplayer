# VisualPlayer

<img src="resources/icon.svg" alt="VisualPlayer" width="112" />

**거의 모든 포맷을 재생하는 가볍고 비영리적인 크로스 플랫폼 미디어 플레이어.**

`Rust · Tauri 2 · Swift` ｜ `Windows · macOS · Linux · Android · iOS`

[English](./README.md) ｜ 한국어 ｜ [日本語](./README.ja.md)

</div>

---

VisualPlayer는 **mpv(libmpv, 내부 FFmpeg)** 로 거의 모든 영상·오디오 포맷을 재생합니다.
다중 재생, 동명 자막 자동 연동, 토렌트·웹·유튜브 링크 파일, 단축키, 터치 제스처를
지원하는 **무료 오픈소스** 앱입니다.

## 설치

[최신 릴리스](../../releases/latest)에서 플랫폼에 맞는 파일을 받으세요.

| 플랫폼 | 파일 |
| --- | --- |
| Windows 10/11 (x64, Arm64) | `VisualPlayer-Setup-*.exe` (커스텀 설치 프로그램) |
| macOS (Apple Silicon / Intel) | `.dmg` |
| Linux | `.deb` / `.rpm` / `.AppImage` |
| Android | `.apk` |

- **Windows:** `VisualPlayer-Setup-*.exe` 실행 후 폴더·바로가기·파일 연결을 선택하고
  설치. NSIS 없는 단일 설치 프로그램이며, 제거는 프로그램 추가/제거에서.
- **macOS:** 응용 프로그램으로 드래그, 첫 실행은 우클릭 → **열기**.
- **Linux:** 패키지 설치 또는 `chmod +x *.AppImage && ./VisualPlayer.AppImage`.
  시스템 `libmpv` + WebKitGTK 필요.
- **Android:** 출처 허용 후 `.apk` 실행.

## 시작하기

1. **미디어 열기** — 창에 파일을 끌어다 놓거나, 우측 상단 **폴더** 버튼 / 시작 화면의
   **파일 열기**를 누르세요.
2. **여러 파일**을 놓으면 대기열에 추가됩니다(첫 파일 재생, 나머지는 재생목록으로).
   재생목록은 **목록** 버튼 또는 `Tab`으로 엽니다.
3. 마지막 위치에서 **자동으로 이어 재생**합니다(설정에서 끌 수 있음).

## 재생 컨트롤

재생 중 자동 숨김되는 컨트롤 바(마우스를 움직이면 표시):

- 재생/일시정지, 이전/다음, 탐색 바와 시간, 볼륨·음소거, 배속.
- **자막** 버튼으로 자막 트랙 전환.
- **조정** 버튼으로 비디오/오디오 도구 패널 열기.
- **스크린샷**, **재생목록**, **미니 플레이어**, **설정**, **전체화면** 버튼.

### 미니 플레이어

**미니 플레이어** 버튼(`t`)은 창을 작게 줄이고 **항상 위**로 고정해, 작업 중에도
화면 한쪽에서 영상이 계속 재생됩니다. 다시 누르면 원래 크기로 복원됩니다.

## 자막

- **자동** — `movie.mkv`를 열면 같은 폴더의 `movie.srt`, `movie.ass`,
  `movie.ko.srt` 등을 자동으로 연결합니다.
- **수동/내장** — 자막 버튼 또는 `c`로 트랙을 전환합니다.
- **싱크·스타일** — 조정 패널에서 **자막 싱크**(`g`/`h`), 설정에서 **크기**를 조절합니다.

## 조정 (VLC 스타일)

**조정** 패널 기능:

- **비디오:** 화면 비율, 회전, 확대/축소/초기화, 디인터레이스.
- **색상 이퀄라이저:** 밝기·대비·감마·채도·색조(+초기화).
- **챕터:** 내장 챕터 이동(`PageUp`/`PageDown`) 또는 목록에서 선택(챕터가 있을 때만 표시).
- **오디오:** 트랙 전환, **오디오 싱크**(`Shift+G`/`Shift+H`).
- **10밴드 오디오 이퀄라이저:** 프리앰프 + 프리셋(록, 팝, 재즈, 클래식, 댄스,
  저음/고음 강화, 보컬, 소프트); `e`로 켜고 끔.
- **자막:** 트랙 전환, 싱크.
- **A-B 반복:** A 지정 후 B 지정으로 구간 반복(`r`).
- **미디어 정보:** 해상도, 코덱, 프레임레이트, 트랙 수.

## 재생목록

- 파일을 열거나 끌어다 놓아 추가하고, 항목을 클릭해 재생, ✕로 삭제합니다.
- **재생목록 열기**로 `.m3u`/`.m3u8` 파일을 불러옵니다(상대 경로는 재생목록 폴더
  기준으로 해석, URL은 그대로 유지).
- **재생목록 저장**으로 현재 목록을 확장 M3U 파일로 저장합니다.

## 터치 제스처 (모바일)

Android(및 iOS)에서 영상 위:

- **좌우 스와이프** — 탐색(목표 시간 표시).
- **상하 스와이프, 오른쪽** — 볼륨.
- **상하 스와이프, 왼쪽** — 화면 밝기.
- **양쪽 더블탭** — −10초 / +10초 이동.

**설정 → 터치 제스처**에서 켜고 끌 수 있습니다.

## 커스텀 파일 포맷

- **`.{비디오}.torrent` + `.{자막}.torrent`** — 비디오 토렌트를 열 때 같은 이름의
  자막 토렌트(`movie.mp4.torrent` ↔ `movie.srt.torrent`)가 있으면 둘 다 받아 자막을
  비디오에 연결합니다. 다운로드 전용.
- **`.webvideo`** — 직접 미디어 URL(한 줄) 또는 `url`·`headers`를 담은 TOML.
- **`.ytvideo`** — 유튜브 링크(한 줄) 또는 `url`·`preferred_resolution`·`title` TOML.

더블클릭하면 바로 재생됩니다.

## 기본 플레이어로 설정

**설정 → 기본 플레이어로 설정**을 누르면 지원하는 모든 비디오·오디오·자막 형식의 기본
앱으로 VisualPlayer가 등록됩니다(OS 방식 사용, Windows는 기본 앱 설정 화면을 엽니다).

언어(English / 한국어 / 日本語), 테마(시스템/라이트/다크), 이어 재생, 하드웨어 디코딩,
다음 항목 자동 재생, 자막 자동 로드·크기, 제스처, 기본 앱 등록, 단축키 목록, 버전 정보.

## 소스 빌드

사전 요구: Rust(stable), Node.js(LTS), 플랫폼별 Tauri 사전 요구사항. 실제 재생에는
**mpv/libmpv**가 필요합니다.

```bash
# Linux 빌드 의존성 (Debian/Ubuntu)
sudo apt-get install -y libwebkit2gtk-4.1-dev libmpv-dev librsvg2-dev libgtk-3-dev

# macOS
brew install mpv

# 순수 코어 테스트 (네이티브 의존성 불필요)
cargo test -p vp-core

# 데스크톱 개발 / 빌드
cd app/main
npm install
npm run tauri dev
npm run tauri build               # 토렌트 지원은 `-- --features torrent`

# Android (Android SDK/NDK 필요)
npm run tauri android init
npm run tauri android build
```

백엔드는 Cargo **워크스페이스**입니다. `vp-core`는 의존성 없는 단위테스트 로직,
`app/main/src-tauri`는 Tauri 바이너리입니다. 프론트엔드는 **Vite + Vanilla TypeScript**
(프레임워크 없음)이며, iOS는 [`app/ios`](./app/ios)의 별도 Swift 앱입니다.
